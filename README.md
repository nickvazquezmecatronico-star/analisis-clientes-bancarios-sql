# Análisis de clientes bancarios con SQL

Proyecto de análisis de datos utilizando MySQL y SQL para modelar clientes, cuentas y transacciones en un entorno financiero.

## Tecnologías
- MySQL
- SQL
- Modelado relacional

## Contenido
- Script de base de datos
- Tablas: clientes, cuentas, transacciones
- Relaciones PK/FK

## KPIs generados
- Dinero total administrado
- Saldo promedio
- Top clientes por saldo
- Ciudad con mayor concentración de dinero
- Tipo de cuenta con mayor movimiento

## Habilidades demostradas
- JOIN entre tablas
- Agregaciones (SUM, AVG)
- GROUP BY
- Análisis orientado a negocio
